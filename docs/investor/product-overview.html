Investor brief · June 2026

FPL Decision Companion

Mobile-first Fantasy Premier League toolkit for UK, European, and North American managers — plan transfers, read the gameweek, and track rivals in one product.

Working codebase title: FPL Squad Viewer · Commercial brand: see Brand Naming Options
Production: https://fpl-squad-viewer.fly.dev · Stack: React SPA + Hono BFF + Postgres cache

Executive summary

We are building a subscription freemium companion to the official Fantasy Premier League game. Users link a public FPL team ID (or sign in to sync drafts and watchlists). The product focuses on weekly decision quality — not replacing FPL, not betting, not daily fantasy.

Problem
Serious managers juggle 4–6 websites before each deadline: live rank, fixtures, price changes, predictions, and transfer notes.
Solution
One fast mobile web app with planning, review, predictions, and league intelligence — priced below the mid-tier incumbents.

Market

  • TAM: ~11M FPL managers worldwide (Premier League official game, 2025/26 season).
  • SAM: ~1–2M “active optimisers” who use third-party tools weekly (Scout, Fix, fpl.team, FPL Review cohort).
  • SOM (Year 1 target): 5,000 paying Premium users × £25 season ≈ £125k gross before payment fees; stretch 15k × blended ARPU £30.

Geography: primary UK; secondary Western Europe and US/Canada English-speaking FPL communities. Single English product, GBP-led pricing with multi-currency checkout.

Technology moat (pragmatic)

  • BFF + Postgres cache — trims official FPL API payloads; frozen GW rows; rate-limited prefetch (ADR 0018).
  • Spec-first delivery — OpenSpec capabilities reduce rework; faster iteration than content-heavy incumbents.
  • Free-data ML path — PRED-09 avoids Opta licence cost at launch; upgrade path when revenue supports data deals.
  • Mobile UX — native-feel pitch UI; guided transfer tour; lower cognitive load than desktop-first Scout.

Go-to-market

  1. Community — r/FantasyPL, FPL Twitter/X, Discord servers; compare threads vs fpl.team / Plan FPL.
  2. Content SEO — “price change tonight”, “FDR calendar”, “gameweek review” landing pages.
  3. Free tier virality — guest squad view via team ID link; league browser for rival scouting.
  4. Seasonal campaigns — GW1 draft assistant; DGW chip push notifications (when live).
