What we will cover (parity roadmap)

These are table stakes for Premium/Pro credibility — users already get them elsewhere:

  • Predicted lineups for all Premier League clubs before each deadline (fpl.team, Fix).
  • Live rank and live mini-league movement on matchdays (fpl.team, LiveFPL, Fix).
  • Transfer solver / multi-GW optimiser (fpl.team, FPL Review MILP, Fix AI).
  • Chip timing assistants — Wildcard, Free Hit, Triple Captain, Bench Boost (fpl.team, Review).
  • Full predicted points without blur (all paid tools).
  • Expanded watchlist — comparable to tracking rivals in Fix / live leagues in fpl.team.

What we will NOT prioritise (and why)

  • Long-form editorial / Scout Picks — Scout’s moat; we stay tool-first.
  • Elite XI live reveals — Fix’s moat; expensive data relationships.
  • WhatsApp communities — LazyFPL model; ops-heavy, low margin.
  • Standalone AI chat SKU — ChatFPL competes on messages; we bundle AI later inside squad context.

Win / loss scenarios

We win
Manager wants one mobile tab for planner + review + prices; hates juggling Scout + LiveFPL + spreadsheet.
We lose
Manager already pays Review + fpl.team for MILP + multi-GW; needs elite reveals or Opta heatmaps today.

Strategic takeaway

Compete on integration and mobile UX at £25 season, not on being the deepest data shop on day one. Close the solver/live gap in H2 2026, then raise Pro ARPU. Avoid naming collision with fpl.team / Scout / Review — see brand-naming-options.pdf.
