# Backlog / Future ideas

Deferred features and ideas, kept here so they are not forgotten. Not committed work — these
get promoted into an OpenSpec change when we decide to build them.

## Later

- **Dedicated desktop design** — the MVP is mobile-first (centered phone-width on desktop);
  a tailored desktop layout is planned. Decided alongside ADR 0006.
- **Multi-language UI (i18n)** — copy is centralized and i18n-ready; add languages beyond English.
- **Team of the Week badge on squad screen** — show a small badge/tag next to a player's kit on
  the squad screen when they were selected in the official Team of the Week for that gameweek.
- **Player detail stats (points breakdown)** — tapping a player opens a detail view showing the
  breakdown of how they earned their points (goals, assists, bonus, clean sheet, etc.), mirroring
  what the official FPL app shows.
- **Live in-match scoring behaviour** — analyse and decide how the app should behave while
  matches are in progress: polling frequency, visual indicators for live players, partial score
  display, and edge cases (substitutions, bonus points not yet confirmed).
- **Gameweek bonus breakdown** — bonus points detail per player.
- **FDR screen (Fixture Difficulty Rating)** — a table showing upcoming fixture difficulty per
  team, rated 1–5 (easy→hard), filterable by next N gameweeks, sortable by attack / defence /
  overall. UI reference and design inspiration: https://fpl.team/fdr/ — the visual style of
  rating cells, colour scale, and filter controls is worth following.
- **Predictions / odds** — clean sheet / xG style projections.
- **AI assistant** — team-aware Q&A (as on fplukraine.com). High priority concept; to be
  scoped separately.
