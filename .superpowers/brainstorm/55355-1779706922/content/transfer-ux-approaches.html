A — Inline Tap-to-Swap

Тап на игрока → он помечается OUT → снизу выезжает дроуэр с отфильтрованными заменами. Замена происходит прямо на питче. Несколько незакрытых OUT — несколько активных слотов.

B — Transfer Slots Panel

Отдельная панель «Замены» с явными слотами OUT→IN. Пользователь добавляет слот, выбирает OUT-игрока из своей команды, затем IN-игрока из пула. Весь план виден целиком.

✓ Максимально явно ✓ Легко видеть весь план — но больше шагов, менее интуитивно на мобильном

C — Wizard (пошаговый)

Три шага: выбери кого продать → выбери кого купить → подтверди. Каждый шаг — отдельный экран/состояние. Самый понятный для новичков, но медленнее для опытных.

✓ Линейный поток ✓ Ясная навигация — но медленнее, больше кранов на мобильном

Можно кликнуть по карточке и написать ответ в терминале. Можно и комбинировать (например, A + элемент B для сводки).
