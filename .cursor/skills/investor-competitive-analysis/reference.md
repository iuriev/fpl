## Net-new angles (validate each release)

| Capability | Competitor gap |
| --- | --- |
| Unified price screen (actual + tonight + squad filter) | Usually split across tools |
| Gameweek review + transfer what-if in mobile app | Review has charts; rarely unified with planner |
| Dual watchlists (managers + players) | Often only one type |
| PRED-09 free-data model + `confidence` | Review/Scout use paid data or opaque ep_next |
| PRED-10 NL rationale in player sheet | Chat tools are ad hoc, not embedded |
| Recovery calendar in fixtures | Partial overlap with fpl.team |

## Not our moat (do not compete head-on)

- Scout long-form editorial / Scout Picks
- Fix elite XI live reveals
- LazyFPL WhatsApp / newsletter community
- Standalone ChatFPL message limits SKU

## Win / loss

**Win:** competitive mini-league manager tired of 4+ tabs; wants mobile plan + review + prices together.

**Lose:** manager already on FPL Review + fpl.team for MILP + multi-GW; needs Opta heatmaps or elite reveals today.
