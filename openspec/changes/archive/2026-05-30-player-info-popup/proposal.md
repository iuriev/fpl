# Proposal: Player Info Popup

## Problem
Users want to see detailed information about a player (price, ownership, upcoming fixtures) without leaving the squad view.

## Proposed Solution
Implement a popup/dialog that appears when a player card is clicked. This popup should show:
1. Current price and ownership percentage.
2. The next 5 upcoming fixtures for the player's team.
3. FDR (Fixture Difficulty Rating) for those fixtures, color-coded for quick assessment.

## Scope
- PlayerCard interactivity.
- Player Info Popup UI.
- Upcoming fixtures data integration.
